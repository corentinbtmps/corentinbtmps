### 🚀 À propos

Je suis **Lead Développeur Full-stack** au sein de l'agence [Human To Computer](https://humantocomputer.com), où je porte l'architecture, la qualité et la cohérence technique des projets — tout en continuant à développer au quotidien sur la **stack TALL** (Tailwind, Alpine, Laravel, Livewire).

- 🧡 **10 ans sur la stack TALL**, 15+ ans à construire pour le web
- 🎨 **Double culture design + dev** — ancien graphiste/webdesigner, pour des interfaces au pixel près et maintenables
- ⚡ Obsédé par la performance : chaque projet est classé **Grade A (GTmetrix)** et **90+ PageSpeed**, même les plus complexes
- 🧱 Co-auteur & mainteneur du **starter kit** interne Laravel/Livewire/Tailwind — ~90 modèles, 85+ composants Livewire, suite Pest, CI GitHub Actions
- 🛠️ Créateur de **[Laravel Vitals](https://laravel-vitals.com)** — audits Lighthouse open source (MIT) directement dans une application Laravel
- 🌐 FR (natif) · EN (professionnel complet) · ES (usage limité)
